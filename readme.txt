Running the application.
========================

1. There are initial text values in the textbox.
2. This initial value can be changed.
3. The application can be tested by either pressing the OK button or selecting from the combobox.
3. To test, select the desire method from the combobox and the output will be display in a listbox.
4. It can also be tested by typing in the textbox and pressing the OK button.
5. If an empty string is sent, an error will thrown at the service site and clicking continue will return an error at the client site.
